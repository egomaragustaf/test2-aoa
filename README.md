## Setup

1. Clone the repo

   ```sh
   git clone https://github.com/egomaragustaf/test2-aoa.git
   ```

2. Go to the project folder

   ```sh
   cd test2-aoa
   ```

3. Install packages

   ```sh
   pnpm i
   ```

4. Run the app locally

   ```sh
   pnpm dev
   ```
